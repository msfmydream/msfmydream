jdk1.8 安装配置

  1. 在ORacle 官网下载 jdk8，并上传到服务器

  2. 在服务器 新建 目录 /usr/lib/java

  3. 解压文件到此目录 tar -zxvf jdk-8u311-linux-x64.tar.gz

  4. 修改环境变量 vim /etc/profile
    文件末尾追加

  5. 刷新配置文件 source ~/.bashrc

nginx1.18.0 安装配置

  1. 下载nginx1.20.1 并上传到服务器 目录

  2. 新建目录 /usr/lib/nginx ，并在此解压

  3. 进入目录 ./configure --prefix=/usr/local/nginx 将nginx安装到/usr/local/nginx 目录

![image-20220110171258816](https://gitee.com/msfmydream/picture/raw/master/image-20220110171258816.png)

缺少pcre 依赖库.....

`apt-get install libpcre3 libpcre3-dev`

![image-20220110171936095](https://gitee.com/msfmydream/picture/raw/master/image-20220110171936095.png)

缺少 zlib 依赖库。。。

Google找到他的网站 [zlib](http://zlib.net/) 下载并在服务器配置安装

![image-20220110172945874](https://gitee.com/msfmydream/picture/raw/master/image-20220110172945874.png)



`./configure --prefix=/usr/local`

`make & make install`

回到 目录 `/usr/lib/nginx/nginx-1.20.1`再次执行`./configure --prefix=/usr/local/nginx`
