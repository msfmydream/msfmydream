c++中优先队列(priority_queue)的用法

发表于 2021-09-09 分类于 语法基础

优先队列定义

定义：priority_queue<Type, Container Functional>

Type 表示队列中的数据类型

Container 表示容器的类型，例如vector，queue 等

Functional 表示比较的方式， 默认是大根堆

优先队列的本质是一个堆来实现的

top 访问队头元素

empty 队列是否为空

size 返回队列中元素的个数

push 插入元素到队尾( 根据队列的定义进行排序)

pop 弹出队头元素

swap 交换内容
