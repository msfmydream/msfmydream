git够用手册

发表于 2021-08-30 更新于 2021-08-31 分类于 杂谈

可分为三个区： 工作区，暂存区，远程仓库

1.在本地进行初始化，建立暂存区， .git 文件存储当前项目的所有版本信息

2.将文件从工作区提交到暂存区

3.查看当前工作区的状态

4.从暂存区恢复文件到工作区

5.查看工作区和暂存区版本的区别

6.查看提交过的历史版本

7.恢复文件到指定的版本

8.若要从本地提交到远程仓库，需要先生成ssh密匙

从本地生成密匙后，到 我的电脑=> 用户 =>用户文件夹 => .ssh（隐藏文件） => id_rsa.pub（存有密匙）

拿到这个文件中的密匙后，进入github登录用户，进入 setting 管理密匙中添加该密匙

9.指定本地暂存区提交的远程仓库

10.从远程仓库克隆代码到本地

11.从远程仓库同步代码到本地
